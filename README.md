## 简介

> [!IMPORTANT]
> 以下内容于2025年11月8日撰写，本软件的实际功能可能会因新版本推送而与简介有所差异。

**田字格生成器**是一个基于 Python + Flask 的 Web 项目。其支持

- 技术栈：`Python`
- 支持的操作系统：Xiaomi Vela JS

## 使用教程

1. 下载本仓库
2. 运行程序目录下的 `install_moudle.py`
3. 运行 `app.py` ，默认端口 `5218`
4. 浏览器访问 `localhost:5218` 或者手动配置反向代理，即可使用。
